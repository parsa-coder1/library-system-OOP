# Library Management System (OOP)

An upgraded library management system built with Python using Object-Oriented Programming (OOP) and modular project structure.

## Features

- Add books
- Show all books
- Search books
- Delete books
- Add users
- Show users
- Search users
- Delete users
- Borrow books
- Return books
- Show borrowed books for users
- Save data using JSON
- Load data automatically
- Input validation
- Cancel operations with `q`

---

## Concepts Used

This project practices:

- Object-Oriented Programming (OOP)
- Classes and Objects
- Methods
- Object relationships
- JSON file handling
- Serialization
- Modular project structure
- Helper functions
- Input validation
- Menu-driven programs

---

## Classes

### Book

Represents a book in the library.

### User

Represents a user and manages borrowed books.

### Library

Main system manager that controls library operations.

---

## Project Structure

- `main.py` → Main program loop
- `models.py` → Classes and core methods
- `helpers.py` → Helper functions
- `storage.py` → Save and load system
- `library_data.json` → Stored data file
